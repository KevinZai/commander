# Social Media Content Strategy

## When to Use
When you need a structured social media content plan — not individual posts, but a strategy with themes, cadence, and a content calendar. Use this for planning, then execute individual posts from the calendar.

## Template

```
Create a social media content strategy for the following product/brand. Focus on {{primary_platform}} with supporting content for {{secondary_platforms}}.

**Product/Brand:**
{{what_it_is}}

**Target audience:**
{{who_we_want_to_reach — role, interests, pain points}}

**Primary platform:**
{{Twitter/X|LinkedIn|Reddit|Instagram|TikTok}}

**Secondary platforms:**
{{list_others}}

**Current presence:**
{{describe_current_following_and_activity — or say "starting from zero"}}

**Goals:**
{{awareness|engagement|traffic|conversions — pick 1-2 primary goals}}

**Step 1: Content pillars**
Define 4-5 content themes that rotate:
1. **Educational** — teach something valuable (tutorials, tips, how-tos)
2. **Behind the scenes** — development process, decisions, challenges
3. **Social proof** — user stories, testimonials, metrics, case studies
4. **Community** — engage with followers, respond to trends, reshare
5. **Product** — announcements, features, demos (max 20% of content)

**Step 2: Content formats per platform**
| Platform | Best formats | Frequency | Best times |
|---|---|---|---|
| Twitter/X | Threads, hot takes, code snippets | 5-7x/week | 9am, 12pm, 5pm ET |
| LinkedIn | Long-form posts, carousels | 3-4x/week | 8am, 12pm ET |
| Reddit | Detailed answers, Show HN style | 2-3x/week | varies by subreddit |

**Step 3: Content calendar (2-week sample)**
| Day | Platform | Pillar | Topic | Format | Hook |
|---|---|---|---|---|---|
| Mon | Twitter | Educational | ... | Thread | ... |
| Tue | LinkedIn | Behind scenes | ... | Post | ... |
| Wed | Twitter | Community | ... | Poll | ... |
| ... | ... | ... | ... | ... | ... |

**Step 4: Engagement playbook**
- How to respond to comments (tone, speed)
- How to handle negative feedback
- Which conversations to join (relevant hashtags, communities)
- Influencers and accounts to build relationships with

**Step 5: Metrics to track**
| Metric | Target | Measurement |
|---|---|---|
| Follower growth | X%/month | Weekly check |
| Engagement rate | X% | Per post |
| Click-through rate | X% | Per link post |
| Conversions | X/month | Attribution tracking |
```

## Tips
- Use the `content-strategy` skill for long-term content planning
- The `marketing-pack` skill generates assets (images, copy) for individual posts
- Consistency beats virality — a steady cadence outperforms sporadic bursts

## Example

```
Create a social media content strategy for the following product/brand.

**Product/Brand:** Claude Code Kit — an open-source Claude Code configuration toolkit
**Target audience:** Developers who use Claude Code, AI-assisted coding enthusiasts
**Primary platform:** Twitter/X
**Secondary platforms:** Reddit (r/ClaudeAI, r/ChatGPTPro), LinkedIn
**Current presence:** 0 followers, starting from scratch
**Goals:** Awareness + community engagement
```
