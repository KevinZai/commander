## How to Use (Start Here)

**You don't need to type anything. Multiple choice.** CCC guides you with menus.

### Step 1: Install (one command)

```bash
npm install -g cc-commander
```

### Step 2: Launch

```bash
ccc
```

### Step 3: Pick what you want to do

```
  What would you like to do?

  ‚ĚĮ Build something new        ‚Üź websites, apps, tools
    Import a project            ‚Üź /xray audit + /makeover
    Create content              ‚Üź blogs, social, emails
    Research & analyze          ‚Üź competitors, markets
    YOLO Mode                   ‚Üź overnight autonomous build
    Autonomous Mode             ‚Üź loops, dispatch, background agents
    Browse CCC Domains          ‚Üź 11 domains, 172+ sub-skills
    Check my stats              ‚Üź cockpit dashboard
```

Use **arrow keys** to navigate. Press **Enter** to select.

### Step 4: Answer a few questions

CCC asks what you need (multiple choice ‚ÄĒ just pick one):

```
  What's the most important outcome?

  ‚ĚĮ Something that works end-to-end
    A solid foundation to build on
    A quick prototype to test the idea
```

### Step 5: CCC does the rest

It dispatches to the best available tool (from 11 vendor packages), tracks the session, and learns from the results.

**No commands to memorize. No flags to type. No config files.** Just answer questions.

| Method | Command | For |
