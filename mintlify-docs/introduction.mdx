> **One plugin. 55 skills. Free forever. Your guided AI PM for every AI coding agent.**

**🖥️ Primary surface: Claude Code Desktop (aka Cowork Desktop).** Install once via Settings → Plugin Marketplace — 55 skills, 17 agents, 8 lifecycle hooks (16 handlers), and 2 credential-free bundled MCP servers (+16 opt-in via `/ccc-connect`) appear in every session automatically.

> Cowork Desktop and Claude Code Desktop are the same app, two UI modes. The plugin works identically in both. Type `/ccc` to open the hub.

CC Commander is a guided AI PM that helps you master Claude Code — or any Claude-compatible agent IDE — instantly. It's not a skill pack. It reads your project state, scores complexity, routes to the right specialist sub-agent, and gets smarter every session. Built by distilling 200+ community sources into one install, with 17 specialist agent personas, a sub-agent architecture built on Anthropic's Claude Agent SDK pattern, and 8 lifecycle hooks that fire automatically around every session.

## Why Free Forever?

<Note>
  **CC Commander is free forever.** No paid tier is planned. No caps. No Stripe.

  Revenue comes from four sources — none of which require you to pay:
  1. **Consulting** (primary) — Kevin's consulting pipeline. The plugin is a trust engine.
  2. **Affiliate links** (passive) — transparent disclosures in `/ccc-connect` (Supabase, Vercel, Neon, etc.)
  3. **Pro Community** (optional) — a paid Discord/community tier may launch post-beta. The plugin stays free.
  4. **GitHub Sponsors** (optional) — for those who want to contribute directly.

  All 55 plugin skills, 17 sub-agents, 8 lifecycle hooks (16 handlers), and 2 credential-free bundled MCP servers (+16 opt-in via `/ccc-connect`) are available to everyone, always.
</Note>
