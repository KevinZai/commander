## Campaign Templates

### Full SEO Audit
1. `seo-context` → capture current state
2. `seo-optimizer` → technical audit
3. `site-architecture` → structure review
4. `backlink-audit` → link profile
5. `search-console` → performance data
6. `seo-dashboard` → unified report
7. Deliver findings + prioritized action plan

### Content-Led SEO Campaign
1. `seo-context` → domain + competitor analysis
2. `content-cluster` → topic cluster strategy
3. `seo-content-brief` → briefs for each page
4. `blog-engine` → set up publishing system
5. `social-integration` → sharing optimization
6. `analytics-conversion` → tracking setup
7. Monitor with `search-console` + `serp-analyzer`

### AI Search Optimization
1. `seo-context` → current AI search presence
2. `ai-seo` → optimize for LLM citations
3. `aaio` → robots.txt, markdown twins, structured data
4. `seo-optimizer` → technical foundation
5. Verify with manual AI search queries

### Programmatic SEO at Scale
1. `seo-context` → target keywords + competitors
2. `bulk-page-generator` → template design + data sourcing
3. `site-architecture` → URL structure + internal linking
4. `seo-optimizer` → technical SEO for generated pages
5. `analytics-conversion` → tracking at scale
