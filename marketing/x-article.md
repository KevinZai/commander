### Tweet 7 (The /init Wizard)

Every project starts with /init -- a "Choose Your Own Adventure" wizard.

It asks one question: what kind of build is this?

[1] QUICK -- under 4 hours
[2] DEEP -- 1-5 days, spec-first
[3] SAAS -- 1-4 weeks, full lifecycle
[4] OVERNIGHT -- 6-12h autonomous

Then it loads the right skills, generates your CLAUDE.md, sets up your tasks directory, and configures settings.json.

No guessing. No manual setup.

### Tweet 13 (The Install)

Want it? One command:

```
curl -fsSL https://raw.githubusercontent.com/KevinZai/cc-commander/main/install-remote.sh | bash
```

Or clone it:

```
git clone https://github.com/KevinZai/cc-commander.git
cd cc-commander
./install.sh
```

It backs up your existing ~/.claude/ before touching anything. Safe to try.
