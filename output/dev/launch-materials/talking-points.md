## Common objections + answers

**"Isn't this just another skill bundle?"**
No — skill bundles are static lists. CC Commander has an intelligence layer (`/ccc-suggest`) that picks for you, named third-party plugins by relevance, and a shared brain between the plugin and the CLI.

**"Why MIT? What's the catch?"**
No catch. Plugin is MIT forever. Hosted MCP is metered (free 1K calls/mo in beta). Pro tier is for unlimited. Plugin works 100% standalone — you can disable the hosted MCP and it still does everything.

**"What about [specific competing plugin]?"**
We integrate with it. See `docs/ECOSYSTEM.md` for our 20 vendor submodules. `/ccc-suggest` recommends competing plugins by name when they're the right tool for your task. Genuinely better together.

**"How is this different from Anthropic's own plugins?"**
Anthropic's knowledge-work-plugins target specific verticals (finance, sales, PM). We're a meta-layer on top — we reason about which plugin (theirs, ours, community) you need right now.

**"Is the intelligence really 'Opus-class'?"**
Yes — `/ccc-suggest` runs on Opus when explicitly invoked. The ambient background ticker uses Sonnet (cheaper, runs every turn). That's the pricing discipline: full reasoning only when you ask.

## Brand voice (keep these)

- **Decisive, emoji-forward, no hedging.** ("🟢 My call: X. Rationale: Y.")
- **Anti-sycophancy.** Say "this is a bad idea because" when it is.
- **Tables over paragraphs** for 3+ options.
- **Always recommend one** — not "A/B/C, you decide". ⭐ the best one.
- **Show, don't tell.** Every claim gets a specific plugin name / skill count / file path.

**Don't use:** "simply", "just", "obviously", "powered by AI", "revolutionary", "game-changing".
**Use:** "one click", "in under 30s", "502+ skills", "Opus-class reasoning", "click-first".
