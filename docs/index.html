Landing page, bug fix, prototype. Direct execute, ship fast. Sonnet-powered, stack-specific skills only.

DEEP 1–5 days

Full feature with spec-first development, TDD red/green/refactor, subagent delegation. Opus-level reasoning.

SAAS 1–4 weeks

Complete platform lifecycle. Scaffold, auth, billing, dashboard, SEO, E2E tests, deploy. Everything from zero to shipped.

OVERNIGHT 6–12 hours

Autonomous batch jobs with checkpoints, error recovery, usage-limit retries, and human gates. Set it and sleep.
