curl -fsSL https://raw.githubusercontent.com/k3v80/claude-code-kit/main/install-remote.sh | bash

### Backwards compatible

Commander is an overlay. It reads your CLAUDE.md and `.claude/` вҖ” never writes to them:

```
CC Commander reads в”Җв”ҖвҶ’ CLAUDE.md, .claude/skills/, .claude/settings.json
CC Commander writes в”Җв”ҖвҶ’ ~/.claude/commander/ (separate)
Your Claude Code в”Җв”Җв”Җв”ҖвҶ’ Unchanged. Works exactly the same with or without Commander.
```
