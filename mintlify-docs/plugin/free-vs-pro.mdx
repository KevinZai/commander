CC Commander has **no paid tier**. No Stripe, no license key, no tier check, no feature gating. Everything is free.

## Everything Is Free

All of the following are free, for everyone, always:

- ✅ 51 plugin skills (all specialist workflows + domain routers + meta + writing/debugging + session/lifecycle + MIT ports + /ccc-upgrade)
- ✅ 17 specialist sub-agent personas
- ✅ 8 lifecycle hook events / 16 handlers
- ✅ 2 credential-free bundled MCP servers (context7 + sequential-thinking) + 16 opt-in via `/ccc-connect`
- ✅ 502+ ecosystem skills across 11 CCC domains
- ✅ CLI with 83 slash commands and 10 themes
- ✅ Knowledge compounding + session memory
- ✅ Intelligence layer (complexity scoring, stack detection)

No caps. No call limits. No mandatory feedback. No upgrade nudges.

## How It Stays Free

CC Commander is a **trust engine, not a SaaS**. Revenue comes from four sources — none require you to pay:

| Source | Type | How it works |
|--------|------|-------------|
| **Consulting** | Primary | Kevin's consulting pipeline. Companies that use CC Commander hire Kevin. |
| **Affiliate links** | Passive | Transparent disclosures in `/ccc-connect` (Supabase, Vercel, Neon, etc.). You only see them if you opt in to the connector skill. |
| **Pro Community** | Optional | A paid Discord/community may launch post-beta — for deeper access to Kevin and the network, not gated features. |
| **GitHub Sponsors** | Optional | For those who want to contribute directly. |

The plugin itself will never be paywalled.

## A Note on "Pro" in Old Docs

If you've seen references to a "Pro tier" at $15/mo or $19/mo, or "1000 calls/mo beta limit" — those were placeholders from an earlier monetization model that was abandoned before launch. The business model is now locked as **free forever**.

There is no upgrade path because there is nothing to upgrade to.

## Commander Hub (v4.1 — Future)

A **Commander Hub marketplace** (80/20 creator rev-share) is planned for v4.1 — think Gumroad for skills. If you build a skill and publish it, you keep 80% of any revenue. CC Commander itself remains free; the Hub is an optional creator layer on top.

This is not a paywall — it's a distribution mechanism for third-party skill creators.

## Install

