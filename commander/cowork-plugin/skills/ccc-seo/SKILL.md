## AI SEO Features (AEO/GEO/LLMO)

This domain specializes in optimization for AI-powered search surfaces:
- **AEO** (Answer Engine Optimization) — structure content for direct answers
- **GEO** (Generative Engine Optimization) — appear in AI-generated summaries
- **LLMO** (Large Language Model Optimization) — improve citation probability in ChatGPT, Claude, Perplexity, Gemini

## Campaign Templates

### Technical SEO Audit + Fix
1. `seo-context` → capture domain state
2. `seo-optimizer` → meta tags, sitemaps, Core Web Vitals
3. `site-architecture` → URL structure + internal linking
4. `search-console` → crawl errors, index coverage
5. `seo-dashboard` → health overview

### AI Search Optimization
1. `seo-context` → baseline
2. `ai-seo` → LLM citation optimization
3. `aaio` → robots.txt for AI, markdown twins, JSON-LD
4. `content-cluster` → topic authority building

### Content Engine Setup
1. `content-strategy` → editorial calendar + content pillars
2. `seo-content-brief` → per-page keyword targeting
3. `blog-engine` → Next.js/Astro blog with MDX + sitemap
4. `analytics-conversion` → track content conversions
