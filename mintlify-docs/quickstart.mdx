## Desktop Plugin (Recommended)

The fastest path to CCC v3.0.0. Two commands inside any Claude Code session:

<Steps>
  <Step title="Add to marketplace">
    ```
    /plugin marketplace add KevinZai/commander
    ```
  </Step>
  <Step title="Install the plugin">
    ```
    /plugin install ccc
    ```
  </Step>
  <Step title="Open the command center">
    ```
    /ccc
    ```
    Arrow keys to navigate. Enter to select. The full menu appears with skill recommendations based on your current project.
  </Step>
</Steps>

**What you get (Free tier):** 15 plugin skills, SessionStart + Stop hooks, full CLI access.

**Upgrade to Pro ($19/mo):** 5 agents, 8 MCP integrations, 4 advanced hooks. [See cc-commander.com/pricing](https://cc-commander.com/pricing)

## Advanced / Legacy Install Paths

These paths are for developers who want full CLI control or prefer staying inside the terminal.

<Tabs>
  <Tab title="CLI (npm)">
    You use `claude` in your terminal and want the full interactive CLI binary.

    ```bash
    npm install -g cc-commander
    ```

    Then launch the interactive CLI:

    ```bash
    ccc
    ```

    **What you get:** Full CLI with tmux split mode, daemon, theme switching, cockpit dashboard, and all 456+ skills.

    **Best for:** Developers who live in the terminal.
  </Tab>

  <Tab title="Slash Commands (curl)">
    You don't want a separate CLI binary. Just skills + commands inside Claude Code sessions.

    ```bash
    curl -fsSL https://raw.githubusercontent.com/KevinZai/commander/main/install-remote.sh | bash
    ```

    Then in any Claude Code session:

    ```
    /ccc
    ```

    Full interactive menu appears. Same features, no extra binary.

    **What you get:** All 456+ skills and commands accessible via `/ccc` prefix.

    **Best for:** Developers who prefer staying inside Claude Code.
  </Tab>
</Tabs>

## Your First 60 Seconds

```
1. /plugin install ccc   → Plugin installs
2. /ccc                  → Main menu (15 skills)
3. Pick a skill          → CCC loads the right context
4. Session ends          → CCC auto-saves lessons
5. Next session          → CCC is smarter about your project
```

No YAML. No API keys. No configuration.

## Verify Your Install

```bash
ccc --test
```

Runs 22-point self-verification. All green means you're ready.

```bash
ccc --status
```

Health check JSON — shows gateway, skill count, session state.
