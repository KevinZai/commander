| `/ccc-build` | `/ccc-build` or "build something" | Guided build wizard ŌĆö web app, API, CLI, or mobile. Dispatches the builder agent with spec |
| `/ccc-review` | `/ccc-review` or "review my code" | Branch audit ŌĆö diff review, security pass, performance check, or full x-ray. Uses reviewer agent |
| `/ccc-ship` | `/ccc-ship` or "ship this" | Pre-flight checks + release + deploy. 22-point gate: TypeScript, tests, secrets, env vars |
