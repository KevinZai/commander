CCC ships **5 specialized agents** (Pro tier). Each agent has a tightly scoped role, a model appropriate to its task, and tool restrictions that match its responsibilities.

## The Fleet

## How delegation works

Skills delegate to agents automatically when the task exceeds their scope:

| Skill | Delegates to | When |
|-------|-------------|------|
| `/ccc:code-review` | `reviewer` | Diff > 500 lines or 4-dimension scored review requested |
| `/ccc:build` | `builder` | After Spec Flow approval |
| `/ccc:research` | `researcher` | Multi-source synthesis tasks |
| `/ccc:night-mode` | `builder` | Autonomous overnight build loop |
| `/ccc:fleet` | `fleet-worker` | Each parallel task spawn |

<Tip>
  Agents inherit the current CCC tier. Pro users get all 5 agents; Free users can use hub skills but agents are Pro-gated.
</Tip>

## Customizing agents

You can tune agent behavior via `~/.claude/commander/config.json`:

```json
{
  "agents": {
    "debugger": {
      "maxTurns": 60,
      "fallbackModel": "sonnet"
    }
  }
}
```

See the [free vs pro](/plugin/free-vs-pro) page for tier details.
