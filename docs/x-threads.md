## Thread 2: 10 Claude Code Power Tricks (12 tweets)

### Tweet 1 (Hook)
10 Claude Code power tricks that changed how I build software.

Most devs use 10% of what Claude Code can do.

These techniques compound.

### Tweet 2 -- Trick 1
1/ Spec-first development with /plan

Never jump straight to code. /plan interviews you, writes a spec, then executes in a fresh session.

Output quality jumps immediately.

### Tweet 3 -- Trick 2
2/ The confidence-check skill

Ask Claude to rate confidence 0-100% across 4 dimensions before starting complex tasks.

Below 70%? Break it down further. Catches hallucinations before they waste 20 minutes.

### Tweet 4 -- Trick 3
3/ Context is milk. Treat it that way.

Claude degrades in the last 20% of its context window.

context-guard auto-warns at 70%. strategic-compact compresses at logical breakpoints.

Fresh context = better output.

### Tweet 5 -- Trick 4
4/ Subagents for parallel work.

Never let one session do 3 independent tasks. Use /spawn:

Quick: 1 agent, focused task
Team: 2-3 related tasks
Swarm: 4-8 for large builds

Each gets fresh context. Zero bleed.

### Tweet 6 -- Trick 5
5/ Operationalize every bug fix.

After fixing a bug:
- Write a reproduction test
- Sweep codebase for siblings
- Update CLAUDE.md
- Capture in lessons.md

Same bug never happens twice.

### Tweet 7 -- Trick 6
6/ Dialectic review for important decisions.

Spawn 3 agents:
- FOR argues for option A
- AGAINST argues against
- Referee synthesizes

Better than asking one session "what do you think?"

### Tweet 8 -- Trick 7
7/ Four-question validation.

After Claude finishes any implementation, ask 4 targeted questions to surface hallucinations.

94% accuracy catching fabricated code, wrong APIs, phantom dependencies.

30 seconds. Saves hours.

### Tweet 9 -- Trick 8
8/ Auto-checkpoint hooks.

auto-checkpoint git-stashes every 10 file edits.

Claude broke 15 files? Roll back in seconds.

Set and forget. Zero config.

### Tweet 10 -- Trick 9
9/ Workflow modes for context switching.

/cc mode design = building UIs
/cc mode saas = shipping features
/cc mode night = overnight migration

Each adjusts behavior, risk tolerance, and auto-loaded skills. One command.

### Tweet 11 -- Trick 10
10/ CLAUDE.md is your highest-ROI file.

Every rule applies to every future session. After 6 months mine has 200+ rules.

Claude almost never makes the same mistake twice.

Instructions compound like interest.
